# 📷 圖片分類網頁應用（React 專案）

## 🌐 中文（Chinese）

這是一個使用 React 打造的網頁應用程式，主要用於展示圖片分類或標註的功能。使用者可以在前端介面上進行圖片選擇、分類、顯示等操作。此專案展示了我對於前端框架 React 的理解與實作能力，並可作為個人作品集使用。

### 🔧 使用技術
- React.js
- JavaScript (ES6)
- CSS / HTML5
- Node.js (Create React App)

### 🚀 如何啟動
```bash
npm install
npm start
```
